# Configure Linux Server
> What can you install
* Static ip
* DNS
* DHCP
* Domain
* SSH
* APACHE
* Openfire
* Rsyslog
* Samba
* NFS
* FTP
* Cockpit
* Asterisk
* Failover

>How to run the script
```sh
$ cd /home/$USER/
$ git clone https://github.com/TFerreira25/configure_server_linux/
$ chmod +x config_server.sh
$ ./config_server.sh or $ bash config_server.sh
```
